# Reprap Kalibrierungs Plugin

DWC / RRF Kalibrierungs-Plugin fuer Flow, Level-Test, PID, Probe-Tests, LED-Logik und Daemon-Module.

## Schnellstart

1. Nimm die Datei aus `release/`:
   - `Kalibrierung-2.1.7.zip`
2. In DWC Plugin-Manager installieren.
3. Reiter `Kalibrierung` oeffnen.

## Projektstruktur

- `release/` Fertige ZIP fuer Installation
- `source/Kalibrierung/` Voller Plugin-Quellcode zum Weiterentwickeln
- `docs/` Rechtliches und Hinweise

## Entwicklung

Der aktive Quellcode liegt in:
- `source/Kalibrierung/`

Wichtige Dateien:
- `source/Kalibrierung/Kalibrierungs.vue`
